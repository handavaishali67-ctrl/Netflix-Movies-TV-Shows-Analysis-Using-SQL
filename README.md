## Overview

This project analyzes Netflixâ€™s movies and TV shows dataset using SQL to understand content distribution, trends, and patterns. The goal is to demonstrate structured data querying, filtering, aggregation, and analytical reasoning using SQL.

---

## Objectives

- Analyze the distribution of Movies and TV Shows on Netflix  
- Identify the most common content ratings across different content types  
- Examine content trends based on release year and duration  
- Analyze country-wise content availability  
- Categorize and filter content using specific conditions and keywords  

---

## Dataset

- **Source:** Kaggle  
- **Dataset Name:** Netflix Movies and TV Shows  
- **Link:** https://www.kaggle.com/datasets/shivamb/netflix-shows/data  

The dataset contains information such as title, type, director, cast, country, release year, rating, duration, and description.

---

## Approach

- Imported and explored the dataset structure  
- Cleaned and standardized columns required for analysis  
- Used SQL queries to filter, group, and aggregate data  
- Applied conditional logic to categorize content  
- Derived insights based on query results  

All analysis was performed using SQL without modifying the original dataset structure.

---

## Key Analyses Performed

- Distribution of Movies vs TV Shows  
- Most frequent ratings for each content type  
- Content count by release year and country  
- Duration-based analysis for movies and TV shows  
- Filtering content based on keywords in descriptions
